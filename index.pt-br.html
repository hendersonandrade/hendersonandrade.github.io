AzureUtils — módulo PowerShell

Cmdlets utilitários práticos para administração, governança, inventário e automação operacional do Azure. Inventário de tags com round-trip para Excel, descoberta de recursos órfãos, verificação de exposição pública e mais — tudo com saída normalizada e pronta para automação.

Open Source PowerShell 7+ Azure PSGallery

AER — Azure Estate Report

Um projeto da TrimTechBR: um módulo PowerShell 7 que conecta ao Azure com suas credenciais, coleta um inventário somente-leitura via Azure Resource Graph e ARM, e gera um relatório HTML rico, multipágina e autocontido — além de exportações em Excel e PDF.
