Soluções open source

Tudo aqui é open source. O AzureUtils é o meu módulo PowerShell pessoal; o AER é um projeto da TrimTechBR construído junto com Henrique Mizael. Mais virão conforme o tempo permitir.

⌘

AzureUtils

Cmdlets utilitários práticos para administração, governança, inventário, troubleshooting e automação operacional do Azure. Não substitui os módulos Az — encapsula a dor do dia a dia em comandos simples, com saída normalizada e pronta para automação.

  • Export / Set-AzureUtilsTagInventory — round-trip de tags via Excel
  • Find-AzureUtilsOrphanResource — discos, NICs, IPs, NSGs e rotas sem uso
  • Find-AzureUtilsEmptyResourceGroup — resource groups vazios
  • Find-AzureUtilsPublicResource — recursos expostos à internet

AER — Azure Estate Report

Um projeto open source da TrimTechBR. Um módulo PowerShell 7 que conecta ao Azure com suas credenciais, coleta um inventário somente-leitura via Azure Resource Graph e APIs REST do ARM, e gera um relatório HTML rico, multipágina e autocontido — além de exportações em Excel e PDF.

A AER tem seu próprio site e hub de documentação dedicados na TrimTechBR.
