AzureUtils

Practical utility cmdlets for Azure administration, governance, inventory, troubleshooting and operational automation. It does not replace the Az modules — it wraps real day-to-day pain behind simple commands with normalized, automation-ready output.

  • Export / Set-AzureUtilsTagInventory — Excel tag round-trip
  • Find-AzureUtilsOrphanResource — unused disks, NICs, IPs, NSGs, routes
  • Find-AzureUtilsEmptyResourceGroup — empty resource groups
  • Find-AzureUtilsPublicResource — internet-exposed resources

AER — Azure Estate Report

A TrimTechBR open-source project. A PowerShell 7 module that connects to Azure with your existing credentials, collects a read-only inventory through Azure Resource Graph and ARM REST APIs, and renders a rich, multi-page, self-contained HTML report — plus Excel and PDF exports.

AER has its own dedicated website and documentation hub at TrimTechBR.
