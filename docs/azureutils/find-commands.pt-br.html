Comandos Find

Cmdlets de descoberta que trazem à tona o que merece sua atenção num ambiente Azure — recursos órfãos, resource groups vazios e recursos expostos à internet pública. Cada um retorna objetos para o pipeline e também imprime uma tabela colorida no console.

Escopo comum

Todos os cmdlets Find-* aceitam -SubscriptionId (padrão: todas habilitadas) ou -ManagementGroupId (mutuamente exclusivos) e emitem objetos para o pipeline, então você pode filtrar, formatar ou exportar.

⚠
Estes comandos usam heurísticas. Sempre revise os resultados antes de excluir ou alterar qualquer coisa — um recurso pode legitimamente bater numa regra.

Find-AzureUtilsOrphanResource

Encontra recursos que parecem órfãos (sem uso / não associados) via Azure Resource Graph e explica cada um numa coluna Reason. Retorna objetos (também mostrados como tabela colorida no console). Todas as categorias são checadas por padrão; restrinja com -Type.

Valor de -TypeÓrfão quando…
Diskdisco gerenciado com diskState = Unattached
NetworkInterfaceNIC sem VM e sem private endpoint
PublicIPIP público sem configuração de IP e sem NAT gateway
NetworkSecurityGroupNSG não associado a nenhuma NIC ou subnet
RouteTableroute table não associada a nenhuma subnet
pwsh — Find-AzureUtilsOrphanResource
# contexto atual, todas as categorias
Find-AzureUtilsOrphanResource

# um management group, apenas duas categorias
Find-AzureUtilsOrphanResource -ManagementGroupId 'PLAT' -Type Disk, PublicIP

# exporta para CSV
Find-AzureUtilsOrphanResource | Export-Csv .\orphans.csv -NoTypeInformation

Find-AzureUtilsEmptyResourceGroup

Lista os resource groups cuja contagem de recursos (de resources) é zero, no escopo escolhido.

pwsh — Find-AzureUtilsEmptyResourceGroup
Find-AzureUtilsEmptyResourceGroup
Find-AzureUtilsEmptyResourceGroup -SubscriptionId $sub1, $sub2

Find-AzureUtilsPublicResource

Traz à tona recursos expostos à internet pública via Azure Resource Graph, com uma coluna Exposure explicando cada achado. Retorna objetos (também tabela colorida no console). Todas as categorias são checadas por padrão; restrinja com -Type.

Valor de -TypeSinaliza
PublicIpendereços IP públicos que estão associados (em uso)
PublicNetworkAccessrecursos com properties.publicNetworkAccess = Enabled (discos gerenciados excluídos — Enabled por padrão e não é exposição real)
StorageOpenstorage com acesso anônimo a blob ou networkAcls.defaultAction = Allow
NsgInternetInboundregras inbound Allow de NSG originadas da Internet (* / 0.0.0.0/0)
pwsh — Find-AzureUtilsPublicResource
Find-AzureUtilsPublicResource
Find-AzureUtilsPublicResource -ManagementGroupId 'PLAT' -Type StorageOpen, NsgInternetInbound
Find-AzureUtilsPublicResource | Export-Csv .\public-exposure.csv -NoTypeInformation
ℹ
Um recurso pode aparecer mais de uma vez quando várias categorias se aplicam. Revise antes de agir.
