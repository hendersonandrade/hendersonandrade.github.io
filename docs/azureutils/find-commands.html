Find commands

Discovery cmdlets that surface things worth your attention across an Azure estate — orphaned resources, empty resource groups and resources exposed to the public internet. Each returns objects for the pipeline and also prints a colored console table.

Common scope

All Find-* cmdlets take -SubscriptionId (default: all enabled) or -ManagementGroupId (mutually exclusive) and emit objects for the pipeline, so you can filter, format or export them.

⚠
These commands use heuristics. Always review the results before deleting or changing anything — a resource may legitimately match a rule.

Find-AzureUtilsOrphanResource

Finds resources that appear orphaned (unused / unassociated) via Azure Resource Graph and explains each in a Reason column. Returns objects (also shown as a colored console table). Categories are all checked by default; narrow with -Type.

-Type valueOrphan when…
Diskmanaged disk diskState = Unattached
NetworkInterfaceNIC with no VM and no private endpoint
PublicIPpublic IP with no IP configuration and no NAT gateway
NetworkSecurityGroupNSG not associated to any NIC or subnet
RouteTableroute table not associated to any subnet
pwsh — Find-AzureUtilsOrphanResource
# current context, all categories
Find-AzureUtilsOrphanResource

# a management group, only two categories
Find-AzureUtilsOrphanResource -ManagementGroupId 'PLAT' -Type Disk, PublicIP

# export to CSV
Find-AzureUtilsOrphanResource | Export-Csv .\orphans.csv -NoTypeInformation

Find-AzureUtilsEmptyResourceGroup

Lists resource groups whose resource count (from resources) is zero, across the chosen scope.

pwsh — Find-AzureUtilsEmptyResourceGroup
Find-AzureUtilsEmptyResourceGroup
Find-AzureUtilsEmptyResourceGroup -SubscriptionId $sub1, $sub2

Find-AzureUtilsPublicResource

Surfaces resources exposed to the public internet via Azure Resource Graph, with an Exposure column explaining each finding. Returns objects (also a colored console table). Categories are all checked by default; narrow with -Type.

-Type valueFlags
PublicIppublic IP addresses that are associated (in use)
PublicNetworkAccessresources with properties.publicNetworkAccess = Enabled (managed disks excluded — Enabled by default and not a real exposure)
StorageOpenstorage with anonymous blob access or networkAcls.defaultAction = Allow
NsgInternetInboundNSG inbound Allow rules sourced from the Internet (* / 0.0.0.0/0)
pwsh — Find-AzureUtilsPublicResource
Find-AzureUtilsPublicResource
Find-AzureUtilsPublicResource -ManagementGroupId 'PLAT' -Type StorageOpen, NsgInternetInbound
Find-AzureUtilsPublicResource | Export-Csv .\public-exposure.csv -NoTypeInformation
ℹ
A resource may appear more than once when several categories apply. Review before acting.
