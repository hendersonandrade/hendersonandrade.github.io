AzureUtils

Cmdlets utilitários práticos para administração, governança, inventário, troubleshooting e automação operacional do Azure. O AzureUtils não substitui os módulos Az — ele encapsula a dor real do dia a dia (inventário em escala, diagnósticos transversais, checagens de governança, correções seguras) atrás de comandos simples, com saída normalizada e pronta para automação.

O que é

O AzureUtils é uma caixa de ferramentas focada em quem opera Azure todos os dias. Onde os módulos oficiais Az entregam os blocos crus, o AzureUtils entrega os resultados — um inventário de tags que você edita no Excel e devolve, uma lista de recursos que parecem órfãos, os recursos expostos à internet pública — cada um atrás de um único comando com saída limpa e amigável ao pipeline.

Convenções

  • Somente PowerShell 7+ (CompatiblePSEditions = Core).
  • Comandos Get-* retornam objetos; comandos Export-* escrevem arquivos e imprimem um relatório no console.
  • Comandos de leitura usam Get-* / Find-*; comandos que alteram suportam -WhatIf / -Confirm.
  • Toda a saída e as mensagens dos comandos estão em en-US.

Requisitos

Os três módulos abaixo são declarados como RequiredModules, então o Install-Module os instala automaticamente.

MóduloObrigatórioFinalidade
Az.AccountsSimResolve o contexto do Azure (Connect-AzAccount).
Az.ResourceGraphSimAlimenta a consulta de inventário (multi-escopo, auto-paginada).
ImportExcelSimUsado por Export-AzureUtilsTagInventory para escrever o .xlsx.
ℹ
O Set-AzureUtilsTagInventory requer adicionalmente Az.Resources para aplicar as tags de volta no Azure.

Instalação

pwsh
# Estável — instala também Az.Accounts, Az.ResourceGraph, ImportExcel
Install-Module AzureUtils -Scope CurrentUser

# Pré-lançamento
Install-Module AzureUtils -AllowPrerelease -Scope CurrentUser

Índice de comandos

ComandoTipoO que faz
Export-AzureUtilsTagInventoryExportInventário de recursos + tags para uma planilha Excel.
Set-AzureUtilsTagInventoryAlteraçãoAplica uma planilha de inventário editada de volta no Azure.
Find-AzureUtilsOrphanResourceFindRecursos que parecem órfãos, com a razão.
Find-AzureUtilsEmptyResourceGroupFindResource groups sem nenhum recurso.
Find-AzureUtilsPublicResourceFindRecursos expostos à internet pública.
